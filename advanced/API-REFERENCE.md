# API Reference

Complete API reference for NeuroLink.

## Core API

### Generate Text

```http
POST /api/generate
```

### Stream Text

```http
POST /api/stream
```

### Provider Status

```http
GET /api/status
```

## MCP Integration

### List MCP Tools

```http
GET /api/mcp/tools
```

### Execute MCP Tool

```http
POST /api/mcp/execute
```

### MCP Server Status

```http
GET /api/mcp/status
```

For complete API documentation, see [API Reference](../sdk/api-reference.md).
